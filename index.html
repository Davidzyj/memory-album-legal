Memory Album

回忆相册的公开支持页面已经准备好了

这是一套可以直接用于 App Store 提审的静态页面。你不需要自建服务器，只需要把 `docs/` 发布到 GitHub Pages、Cloudflare Pages、Netlify 或 Vercel。

适用于 App Store Connect 支持 HTTPS 公网访问 可直接托管静态文件
查看隐私政策 查看支持页面

你现在最需要做的事

1. 补齐联系信息

先把支持邮箱替换成真实邮箱。没有服务器也没关系，`QQ 邮箱`、`Gmail`、`Outlook` 都可以作为支持邮箱使用。

2. 发布到静态托管

推荐优先用 GitHub Pages。如果你把当前仓库推到 GitHub，并把 Pages 源设置为 `main` 分支的 `/docs` 目录，这套页面就会自动变成公网 HTTPS 链接。

页面文件位于 `docs/memory-album/`。如果你后面换成自定义域名，只需要保持 `privacy/` 和 `support/` 两个页面仍然可访问即可。
